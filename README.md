## Intro

Este é um projeto base apenas para adição e exibição de livros para uma API em NodeJs. Template está bem básico inicialmente e não contém paginação ainda.

This is a base project just for creation and exhibition of books for a NodeJs API. Page template is pretty basic and still don't have a pagination feature.
